Memory Game

Instructions:

This is a basic game of memory solitaire. Flip over any two cards. If they are a match then they are removed from the board. If they are not a match they are flipped back. The game is won when all cards are removed fromt he board.

To use with a pointing device, simply click or tap the cards.

To use with a keyboard, there are two methods of input. First, you may navigate between the cards with the TAB key and use SPACE or ENTER/RETURN to flip the card over. You may also use the number keys to select a row then column. Use the key 'f' to flip the highlighted card over and the key 'c' to clear your selection.

This game was only tested for use using Chrome in Windows 10 and may not be cross-platform compatible.
